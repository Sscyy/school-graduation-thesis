\section{选题背景}

癌症是当今世界威胁人类生命健康的重大疾病之一。根据《2020年全球癌症统计》报告，2020年全球新发癌症病例约1930万例，死亡病例约990万例，预计到2040年全球年新增癌症患者将达到2840万例，相较于2020年增幅约47\%\cite{Sung2021}。癌症的早期发现与干预是改善患者预后的关键因素。目前，由于尚无完全可靠的恶性肿瘤特异性生物标志物，常规病理检查仍被公认为癌症临床诊断的"金标准"\cite{Rubin2008}。常规病理检查中组织病理图像和显微镜下的组织细胞图像记录了细胞和组织在病理状态下的详细信息。其中，细胞核的大小、形状、数目以及空间分布特征与肿瘤的分型、分期和恶性程度存在直接关联，是病理医生进行定性分析和预后评估的核心形态学依据。准确识别和分割图像中的细胞核，对于疾病的诊断以及后续治疗具有重要意义。

随着高分辨率扫描技术的普及，数字病理扫描仪可将玻璃切片转化为数十亿像素级别的全视野数字切片图像（WSI），推动了数字病理学的快速发展\cite{Niazi2019}。然而，面对包含数以万计细胞的超大视野切片，人工逐一识别和分析细胞核不仅效率低下、劳动强度极大，且分析结果受病理医生主观经验影响明显，难以保证结果的一致性和可重复性。因此，利用计算机辅助诊断（CAD）技术构建自动化的细胞核分析系统已成为迫切需求\cite{Xing2016}。

在CAD系统所涵盖的各类任务中，细胞核实例分割是最基础、最关键的环节之一。该任务要求从病理图像中精确检测并分割出每一个独立的细胞核实例，为后续的核形态计量分析、细胞计数以及肿瘤微环境量化等下游任务提供必要基础\cite{Hayakawa2021}。然而，病理图像中细胞核的自动化实例分割面临多重复杂挑战：染色批次与扫描设备的差异导致细胞核边界往往较为模糊；恶性肿瘤组织中细胞核密度大、相邻实例高度粘连；不同器官类型和病理状态下细胞核形态差异显著，模型的跨域泛化能力难以充分保障\cite{Jiao2022}。

\section{研究问题}

现有主流细胞核实例分割方法存在以下两个尚未充分解决的核心问题：

\textbf{问题一：多任务分支间缺乏特征交互，任务互补性未被利用。}HoVer-Net的NP分支（语义分割）与HoVer分支（距离回归）完全独立运行，两个任务的互补信息无法共享。HoVer分支预测的空间位置信息有助于NP分支更精确地定位核边界，NP分支的分割置信度可指导HoVer分支聚焦于核内像素的距离回归，但现有框架缺乏任何跨分支信息交互机制，任务互补性被白白浪费。

\textbf{问题二：边界感知不足，跨域泛化能力受限。}不同医院、不同染色批次的病理图像在颜色分布和纹理特性上差异显著，模型容易过拟合训练集的外观特征。现有方法的跳跃连接直接传递编码器特征，缺乏对细胞核边界几何信息的专门强化，导致模型在跨数据集场景下性能显著下降。由于细胞核的几何形状（圆形/椭圆形）在不同染色条件下是相对稳定的，充分利用边界几何信息有望提升模型的跨域稳定性。

基于上述问题，本研究提出的核心研究问题为：\textbf{如何以SAM 2.1强大的预训练特征为基础，设计实现多任务分支双向信息交互的增强模块，并通过显式边缘引导强化模型的边界感知能力，从而在提升分割精度的同时改善跨域泛化性能？}

\section{理论意义}

本研究以SAM 2.1 Hiera编码器输出的多尺度FPN特征为核心条件，在HoVer-Net框架基础上开展系统性改进，具有以下理论意义：

\begin{itemize}
  \item 提出双向分支增强（BBE）机制，验证了在多任务分割框架中通过跨分支特征交互提升任务协同性的可行性与有效性，为多任务特征利用策略的设计提供了实证依据；
  \item 通过引入显式边缘几何监督，探索了在特征层面主动强化几何不变性信息的方法，为提升模型跨染色域泛化能力提供了一种轻量化的思路；
  \item 系统验证了以分割任务为目标的大规模预训练模型（SAM 2.1）相较于以分类任务为目标的预训练（ImageNet-ResNet50）在病理图像实例分割中的优越性，为骨干网络预训练策略的选择提供了参考。
\end{itemize}

\section{实用价值}

本研究具有以下直接的临床应用价值：

\begin{itemize}
  \item 实现细胞核的自动化、高精度实例分割，可显著降低病理科医生在定量分析环节的工作负担，提升诊断效率；
  \item 精确的细胞核分割结果可为肿瘤微环境量化分析、肿瘤浸润淋巴细胞（TILs）密度统计等下游任务提供可靠基础，辅助免疫治疗疗效预测；
  \item 边缘引导泛化机制使模型在不同医院、不同扫描设备的切片上均能保持较稳定的性能，提升了算法的实际可部署性。
\end{itemize}

\section{基于传统图像处理的细胞核分割方法}

细胞核自动化分割的研究历史可追溯至上世纪60年代，早期工作以手工设计特征与规则驱动的分割算法为主。根据所依赖的图像特征与分割策略的不同，传统方法大致可归纳为阈值法、分水岭算法、主动轮廓模型以及聚类方法四类。

阈值法是最早用于细胞核分割的技术手段，其核心思想是根据图像的灰度分布设定一个或多个分割阈值，将像素划分为前景（细胞核）与背景两类，进而结合形态学操作获得分割结果。Otsu等人\cite{Otsu1979}提出的最大类间方差阈值选取方法是这类方法的代表，通过最大化前景与背景的类间方差自动确定全局阈值，在图像对比度较高时效果良好。然而，单一全局阈值难以应对染色不均匀的病理图像，为此研究者进一步引入了局部自适应阈值策略。Lu等人\cite{Lu2012}结合混合形态学重建操作，设计了一种自适应局部阈值方案，通过抑制细胞核区域的灰度波动以提升分割稳定性；Wienert等人\cite{Wienert2012}则将全局阈值与轮廓拟合相结合，在多种组织类型的细胞分割任务上均取得了较好的效果。总体而言，阈值法实现简单、计算高效，但对染色差异及复杂背景的适应性较弱，且仅依赖阈值操作难以有效分离粘连成团的细胞核簇。

分水岭算法将图像视为具有山丘与谷地的地形曲面，通过在种子标记点处模拟"洪水漫灌"的过程逐步扩展区域边界，进而实现实例的分离\cite{Vincent1991}。Malpica等人\cite{Malpica1997}最早将形态学分水岭引入细胞核聚集体的分割，证明了该算法在区分紧密排列细胞核方面的可行性，但随之而来的过分割现象也被指出。针对这一问题，Yang等人\cite{Yang2006}提出基于条件腐蚀操作改进实例标记的提取方式，通过更精准的种子生成有效缓解了过分割情况；Cheng等人\cite{Cheng2008}则借助形状标记与标记函数联合约束分水岭的分割过程，在细胞核团簇的处理上获得了更为精细的边界。然而，分水岭算法对种子标记的质量高度敏感，标记位置的偏差或噪声响应均会直接影响最终分割精度，在图像质量较低的病理切片中尤为明显。

主动轮廓模型通过定义一个可形变的闭合曲线，依据由图像特征与形状约束共同构成的能量泛函驱动曲线不断迭代演化，直至收敛至目标轮廓\cite{Bamford1998}。Molnar等人\cite{Molnar2016}将细胞核的椭圆形状先验知识融入能量函数，使曲线在演化过程中保持形态约束，从而显著改善了重叠细胞核的分割效果；Ali等人\cite{Ali2012}则设计了一种将区域、边界与形状信息统一纳入能量框架的主动轮廓模型，在处理组织病理图像中多个相互遮挡的细胞核时展现出较强的鲁棒性。尽管如此，主动轮廓模型的性能对初始曲线的位置与形状较为敏感，当初始轮廓与真实目标偏差较大时，能量泛函容易陷入局部极值，导致曲线收敛至错误位置。

基于聚类的分割方法将图像中的像素点按颜色、灰度或纹理等特征的相似性进行分组，从而实现目标区域的自动划分，无需提前指定细胞核的几何形状。Kothari等人\cite{Kothari2009}利用K-means算法对细胞核前景像素进行聚类，再结合凹度检测与椭圆拟合对粘连区域进行细化分离；Arif等人\cite{Arif2007}引入流形学习对细胞核形状进行建模，在K-means聚类基础上实现了更准确的实例划分；Zhou等人\cite{ZhouX2005}则采用模糊C均值聚类对H\&E染色图像中的细胞核区域进行分割。然而，聚类方法缺乏对细胞核边缘的先验建模，无法精确定位实例边界，在图像噪声较大或细胞形态差异显著的情况下分割质量明显下降。

上述传统方法在细胞核分割研究的早期阶段发挥了重要作用，但均存在共同的局限性：方法设计高度依赖领域先验知识和人工调节的参数，特征表达能力有限，难以同时应对病理图像中边界模糊、染色差异大以及细胞核密集粘连等多种复杂情形，在跨器官、跨批次的数据上泛化能力较弱。随着深度学习技术的兴起，基于大规模数据驱动的特征自动学习方法逐渐取代传统手工特征设计，成为细胞核分割领域的主流研究范式。

\section{基于深度学习的细胞核实例分割方法}

深度学习的快速发展为细胞核实例分割带来了革命性突破。相较于传统方法，深度神经网络能够从大规模数据中端到端地学习层次化的图像特征表示，在应对染色不均、边界模糊及细胞核密集粘连等复杂情形时展现出显著优势。根据方法框架的差异，现有深度学习方法大致可分为基于目标检测的方法和基于语义分割的方法两类。

基于目标检测的方法将实例分割分解为目标定位与像素级分割两个子步骤。两阶段方法的典型代表是Mask R-CNN\cite{He2017mask}，该方法在Faster R-CNN的区域提议网络基础上增加了掩码预测分支，通过ROI Align操作实现特征图与候选区域的精确对齐，进而为每个检测框内的目标生成像素级分割掩码。Mask R-CNN在自然图像实例分割中表现出色，但在病理图像场景下，矩形检测框容易同时包含多个相邻细胞核，造成特征相互干扰，分割精度受限。针对密集小目标的特点，StarDist\cite{Schmidt2018}放弃了矩形检测框的形式，转而以星形凸多边形拟合细胞核形状：网络预测每个前景像素沿一组预定义方向到对应实例边界的距离，结合非极大值抑制生成最终的实例掩码，在密集细胞核场景下具有明显优势。

基于语义分割的方法以像素级分类为出发点，结合多任务学习与后处理机制实现实例区分，是当前细胞核分割领域更为主流的技术路线。U-Net\cite{Ronneberger2015}凭借其编解码结构与跳跃连接机制，在保留多尺度空间信息的同时实现了高效的特征重建，成为医学图像分割领域最具影响力的基础架构，并衍生出大量针对病理图像的改进变体。

为应对密集粘连细胞核难以分离的问题，研究者广泛引入边界感知的多任务学习框架。DCAN\cite{Chen2016dcan}通过双解码器分支分别预测细胞核区域掩码与实例边界概率图，利用两者的互补关系在后处理阶段分离相邻实例；BES-Net\cite{Oda2018}在此基础上引入双路解码与自适应加权损失函数，进一步增强了边界区域的检测精度；CIA-Net\cite{ZhouY2019}则在分割分支与轮廓分支之间构建了双向特征交互模块，同时设计了平滑截断损失函数以抑制标注噪声对训练的干扰，两个分支的协同优化显著提升了整体分割质量。

在上述多任务框架的基础上，HoVer-Net\cite{Graham2019}提出了一种更为精细的实例区分机制。该方法设计了三个并行的预测分支：核分割（NP）分支输出二值前景掩码，水平垂直（HoVer）分支预测每个前景像素到其所属核质心在水平与垂直方向上的坐标偏移，核分类（NC）分支对各实例进行细胞类型预测。在后处理阶段，利用HoVer距离图经Sobel算子求梯度，结合标记控制的分水岭算法实现对粘连细胞核的精确分离，同时完成实例分割与细胞分类两项任务。HoVer-Net在多个公开病理图像数据集上取得了优异性能，已成为该领域广泛采用的基准模型。在此之后，CDNet\cite{He2021cdn}以向心方向图替代坐标偏移图描述核内空间关系，通过方向梯度实现更精细的边界定位；Triple U-Net\cite{Zhao2020}则利用H\&E染色中苏木精通道的光学先验，构建三路并行特征聚合结构，从颜色先验角度强化了核分割的准确性。

近年来，Transformer凭借自注意力机制在全局上下文建模方面的优势，逐渐被引入细胞核分割领域。Swin Transformer\cite{Liu2021swin}通过分层局部窗口注意力与移动窗口策略，在兼顾计算效率的同时捕获了多尺度上下文信息，成为当前视觉主干网络的重要选择。TransNuSeg\cite{He2023transnu}提出了一种轻量级多任务Transformer架构，通过共享编码器同时完成细胞核分割与分类，在降低参数量的同时保持了较高的分割精度。CellViT\cite{Horst2024}将Vision Transformer与HoVer-Net框架深度融合，以ViT强大的全局特征表达能力替换卷积编码器，在多个数据集上大幅刷新了分割性能。

尽管上述方法持续推进了细胞核分割的技术边界，但在特征利用方式上仍以单一编码器特征统一分发至各解码分支为主，未能针对不同预测任务（语义掩码预测与空间距离回归）在特征粒度需求上的差异进行针对性设计，多尺度特征的潜力尚未得到充分挖掘。

\section{大规模预训练分割基础模型}

随着Vision Transformer（ViT）\cite{Dosovitskiy2021}的提出，以大规模数据预训练与Transformer架构为核心的"基础模型"范式逐渐在计算机视觉领域确立起主导地位。这类模型通过在海量多样化数据上进行预训练，形成了强大且可迁移的通用视觉表征，推动研究者开始探索将其适配至各类下游分割任务的可能性。

在此背景下，Meta AI提出的SAM（Segment Anything Model）\cite{Kirillov2023}成为通用图像分割领域的标志性工作。SAM采用ViT作为图像编码器，配合支持点、框、掩码等多种形式的提示编码器以及轻量级掩码解码器，在包含超过10亿个标注掩码的SA-1B数据集上进行训练，在自然图像的零样本分割任务中展现出卓越的通用性与灵活性。

然而，SAM直接应用于医学图像时面临明显的域偏移问题。Huang等人\cite{Huang2024}对SAM在多种医学影像模态上的分割能力进行了系统评估，指出其性能在不同任务间波动显著，在细小结构和边界模糊目标上尤为薄弱，且高度依赖提示输入，难以直接部署于全自动分析场景。为提升SAM在医学领域的适用性，Ma等人\cite{Ma2024}将SAM在覆盖多种模态的大规模医学图像数据集上进行全量微调，得到MedSAM，使其能够更可靠地处理医学图像分割任务。针对病理图像场景，张朝君等人探索了以可学习结构对SAM进行轻量微调、将其转化为高质量伪标签生成器的方案，在边界框弱监督下实现了面向细胞核分割的有效迁移。Shui等人\cite{Shui2025}则构建了一种基于提示驱动的细胞核实例分割框架，将SAM的提示机制与显微图像分析相结合，在跨域适应和少样本场景下展现出一定的灵活性。

在SAM基础上，Meta AI进一步推出了SAM~2\cite{Ravi2024}，引入了层次化视觉Transformer（Hierarchical ViT，Hiera）作为图像编码器。Hiera通过多阶段下采样自然输出多个尺度的特征图，并以特征金字塔网络（FPN）的形式提供从高分辨率细粒度特征到低分辨率粗粒度语义特征的完整多尺度表示，显著丰富了编码器对下游任务的特征供给能力。在此基础上发布的SAM 2.1进一步优化了模型的训练策略与泛化性能，为将其应用于病理图像中的细胞核精细分割提供了更为坚实的特征基础。

然而，现有将SAM引入细胞核分割的工作，通常将编码器的输出特征统一输入单一解码路径，未能充分发挥多尺度特征在应对不同预测任务时的差异化优势。事实上，语义掩码预测更依赖包含丰富类别信息的粗粒度高层特征，而空间距离回归则对保留精细位置细节的细粒度低层特征更为敏感。如何针对这一差异，设计面向多任务分支的自适应特征分配机制，是充分释放SAM 2.1多尺度特征潜力的关键，也是本文后续工作的核心出发点之一。

\section{基于双向分支增强的细胞核实例分割}

利用SAM 2.1替换HoVer-Net架构中的Encoder部分，在此基础上设计双向分支增强模块（BBE），解决NP分支与HoVer分支特征孤立、缺乏任务间协同的问题。

HoVer-Net的NP分支（语义分割）与HoVer分支（距离回归）在现有实现中完全独立运行：HoVer分支预测的空间位置信息可帮助NP分支更准确地定位核边界，NP分支的分割置信度信息可指导HoVer分支聚焦于核内像素的距离回归，但现有框架对这种任务互补性无任何利用。

BBE模块在两个分支完成第一层解码（$u_3$）之后，以双向SE风格的通道注意力机制实现互相调制：
\begin{equation}
  g_{NP} = \mathrm{Sigmoid}\bigl(\mathrm{FC}(\mathrm{GAP}(F_{HV}))\bigr), \quad
  F_{NP}^{*} = F_{NP} + F_{NP} \odot g_{NP}
\end{equation}
\begin{equation}
  g_{HV} = \mathrm{Sigmoid}\bigl(\mathrm{FC}(\mathrm{GAP}(F_{NP}))\bigr), \quad
  F_{HV}^{*} = F_{HV} + F_{HV} \odot g_{HV}
\end{equation}
其中$\mathrm{GAP}$为全局平均池化，$\mathrm{FC}$为全连接层（通道压缩比$r=8$）。两个方向的门控权重独立学习，最后一层线性层零初始化，保证训练初期等同于恒等映射。计算复杂度为$\mathcal{O}(C^2)$，适合高分辨率特征图。

损失函数沿用HoVer-Net原版设计：
\begin{equation}
  \mathcal{L}_{total} = \mathcal{L}_{NP} + \mathcal{L}_{HV}
  = (\mathcal{L}_{CE} + \mathcal{L}_{Dice}) + (\mathcal{L}_{MSE} + \mathcal{L}_{MSGE})
\end{equation}

实验在PanNuke数据集三折交叉验证上评估，设置ResNet50基线（A1）、SAM 2.1无BBE（A2）、SAM 2.1+BBE（A3）三组对比，通过消融实验验证BBE的独立贡献。

\section[基于边缘引导分支与边缘感知跳跃连接的高泛化性细胞核分割]{基于边缘引导分支与边缘感知跳跃连接的\\高泛化性细胞核分割}

细胞核的几何形状（圆形/椭圆形）在不同染色条件下具有相对稳定性，而颜色纹理随染色方案大幅变化。现有方法的跳跃连接直接传递编码器特征，未专门强化边界几何信息，导致模型在跨数据集场景下性能下降。

\textbf{边缘引导分支（Edge Branch）}

在训练阶段引入额外的边缘预测解码头，利用拉普拉斯算子从实例掩码标注中自动提取边缘真值（无需额外人工标注），以Dice Loss进行监督约束，强迫模型在训练中主动学习细胞核轮廓的几何形态，降低对染色外观的过度依赖。推理阶段不使用此分支，不改变推理速度和输出格式。

\textbf{边缘感知跳跃连接（EA-Skip）}

对SAM 2.1编码器Level 0细粒度特征（$256\times256$，空间细节最丰富）进入跳跃连接之前，施加可学习边缘感知增强模块（EAM）。EAM采用双路结构：Branch A为深度可分离$3\times3$卷积（初始化为拉普拉斯算子形式，主动提取高频边缘成分），Branch B为$1\times1$卷积保留低频语义成分，两路相加后归一化输出，向解码器主动传递更丰富的边界信息。
\begin{equation}
  \mathcal{L}_{total} = \mathcal{L}_{NP} + \mathcal{L}_{HV} + \lambda \cdot \mathcal{L}_{Edge}
\end{equation}

其中 $\lambda$ 为边缘损失权重，通过消融实验确定最优值。

实验在PanNuke上验证单数据集精度，核心实验为跨数据集泛化测试（MoNuSeg训练，CPM-17和TNBC测试），设置四组消融对比（$\pm$Edge Branch，$\pm$EA-Skip）。


\section{SAM2.1及双向分支增强实验结果}

为验证SAM2.1作为Encoder以及双向分支增强方法的有效性，本文在PanNuke数据集上开展了初步实验验证，以fold1+fold2为训练集（共5928张$256\times256$图像），fold3为测试集（共2656张）。训练共50个epoch，优化器采用AdamW，批大小为8，SAM 2.1编码器学习率设为$10^{-5}$（保护预训练特征），解码器及其余模块学习率设为$2\times10^{-4}$。评价指标包括像素级DICE系数、实例级聚合Jaccard指数（AJI）以及检测质量（DQ）、分割质量（SQ）、全景质量（PQ）。

实验设置三组对比方法，验证SAM 2.1编码器替换与BBE模块各自的独立贡献：

  \item \textbf{A2}：替换为SAM 2.1编码器，不启用BBE模块；

\end{itemize}

\chapter{论文的创新性尝试}
%% ============================================================

\textbf{创新点一：首次将SAM 2.1 Hiera编码器的多尺度FPN特征引入HoVer-Net双分支解码框架。}

现有方法中，CellViT\cite{Horst2024}将SAM encoder与U-Net解码器结合，但使用的是SAM 1（单尺度$64\times64$特征）；HoVer-Net原版\cite{Graham2019}及其改进版均使用ResNet50单尺度编码器。本研究首次将SAM 2.1 Hiera Base+编码器与HoVer-Net经过充分验证的HoVer-map双分支解码框架结合，SAM 2.1内置FPN天然输出三层真多尺度特征，为后续多尺度路由机制的设计提供了独有的特征条件。

\textbf{创新点二：提出双向分支增强模块（BBE），实现NP与HoVer分支的跨任务特征交互。}

现有细胞核分割方法（包括HoVer-Net及其改进版本）中，多任务分支均独立运行，缺乏跨分支信息交互。本研究设计BBE模块，在两个分支完成第一层解码后，以SE风格双向通道注意力机制实现互相调制：HoVer分支的全局语义统计为NP分支生成通道权重，NP分支的分割置信度为HoVer分支生成通道权重，两个方向独立学习，残差连接保证训练稳定性。消融实验表明，BBE在全部评价指标上均带来一致性提升，DQ改善最为明显（+0.4\%），证明双向交互有效减少了漏检并提升了分割质量。

\textbf{创新点三：提出协同的边缘引导机制（Edge Branch + EA-Skip），通过双层次边界几何强化提升跨域泛化能力。}

从监督信号与特征增强两个层面协同引入显式边缘引导：Edge Branch以无额外标注成本的方式引入边缘几何监督，强迫模型学习染色不变的核轮廓形态；EA-Skip通过拉普拉斯初始化的可学习卷积在Level 0细粒度跳跃连接中主动强化边缘响应，使解码器直接接收更丰富的边界信息。与仅通过损失函数施加边缘约束的方案相比，EA-Skip在特征层面实现了主动的边缘信息注入，是本文在特征工程层面的独立创新。

\section{研究难点}

难点一：BBE门控权重的训练稳定性与有效性验证。BBE模块的最后线性层采用零初始化以保证训练初期为恒等映射，但双向门控机制能否在训练过程中稳定收敛并学习出有意义的分支间依赖关系，需要通过消融实验验证。若任一方向的门控权重退化为近零值（即单向退化），则需进一步分析两个分支互信息的不对称性。

难点二：边缘损失权重$\lambda$的平衡。Edge Branch引入的$\mathcal{L}_{Edge}$与主任务损失之间的权重比例直接影响模型的最终性能。权重过大可能导致主任务精度下降，权重过小则边缘监督效果不显著。需要通过系统的消融实验确定最优$\lambda$值，并分析其在单数据集精度和跨数据集泛化之间的权衡效果。

难点三：跨数据集泛化实验的公平性控制。跨数据集实验（MoNuSeg训练$\rightarrow$CPM-17/TNBC测试）涉及不同数据集的预处理标准、图像分辨率和标注规范差异，需要严格控制实验变量，确保对比实验的公平性，避免因数据处理不一致导致结论偏差。

\section{预期成果}

\begin{itemize}
  \item 在PanNuke数据集三折交叉验证上，本文方法（第三章）相较于HoVer-Net ResNet50基线，指标有明显提升；
  \item 在跨数据集泛化实验（第四章）中，相较于第三章基线方法，CPM-17和TNBC测试集上的AJI和PQ指标均有显著提升，验证边缘引导机制对泛化能力的改善效果；
  \item 消融实验结果清晰说明BBE、Edge Branch、EA-Skip三个模块各自的独立贡献。
\end{itemize}

      \item A1：HoVer-Net baseline（ResNet50，无分类分支）：\textbf{PQ=0.6278，AJI=0.6689}；
      \item A2：SAM 2.1 encoder接入（无BBE）：\textbf{PQ=0.6592（+3.1\%），AJI=0.6909}，验证了SAM 2.1预训练特征相较于ResNet50的显著优势；
      \item A3：SAM 2.1 + BBE双向分支增强：\textbf{PQ=0.6614（+0.2\%），AJI=0.6921}，全部指标一致提升，验证了BBE模块的有效性。
    \end{itemize}

  \item \textbf{待完成工作：}Edge Branch及EA-Skip实现与跨数据集泛化验证、完整论文撰写。
\end{enumerate}
